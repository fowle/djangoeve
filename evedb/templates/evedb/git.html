Работа с Git и полезные материалы

Тут будем хранить список всего того, что может пригодиться в работе на EVE DB.

Команды Git

  • git log - лог действий
  • git status - проверка текущего статуса проекта
  • git add --all - подготовка всех изменений к отправке
  • git commit -m "комментарий" - фиксирование сделанных изменений
  • git push origin - отправка фиксированных изменений в репозиторий
  • git pull origin - получений изменений из репозитория
Репозиторий на GitHub - https://github.com/fowle/djangoeve.git
